# 앱인토스 AI 계약서 요약기 — 사전 조사 리포트
생성일: 2026-03-15

## 핵심 요약 (3줄)
1. 앱인토스는 광고(IAA), 인앱결제(IAP), 토스페이 등 다양한 수익화를 공식 지원하며, 현재 서비스 수수료 면제 중이고 개발/마케팅 인프라를 무상 제공한다.
2. 국내 계약서 분석 AI 서비스는 전부 B2B/고가 모델이며, B2C 개인용 간편 분석 앱은 시장에 없다. 앱인토스 내 법률/계약 서비스도 전무하여 선점 기회가 크다.
3. 계약서 이미지를 Claude API로 전송하는 것은 개인정보 국외이전에 해당하므로 동의 절차, 마스킹 기능, 면책 문구가 필수이며, "일반 정보 제공" 수준으로 기능을 한정해야 변호사법 리스크를 회피할 수 있다.

---

## 1. 앱인토스 플랫폼 정책

### 외부 광고 허용 여부
- **인앱광고(IAA) 공식 허용**: 구글 애드몹 기반, 전면형/보상형/배너 3가지 유형 지원
- 광고 수익 정산: 월 단위, eCPM 기준 (예: eCPM 60,000원 = 노출 1회당 약 60원)
- **다크패턴 엄격 금지**: 진입 시 즉시 광고, 예상치 못한 광고, 닫기 방해 등 불가
- 외부 홍보 시 "토스 공식 파트너사/입점사/제휴사" 표현 금지, 검수 필수

### 수수료 정책
| 항목 | 수수료율 | 비고 |
|------|---------|------|
| 서비스 수수료 | **현재 0%** | "지금은 서비스 수수료가 없어요" (면제 기간 종료일 불명확) |
| 인앱광고(IAA) | 정상 15%, 테스트기간 0% | 구글 애드몹 기반 |
| 인앱결제(IAP) | 토스 5% + 앱마켓 15~30% = **총 20~35%** | 디지털 콘텐츠/서비스용, 결제 범위 400~140만원 |
| 토스페이 | **4.3%** (VAT 별도) | 실물 상품/서비스 판매용, 가맹점 등급별 차등 |

### 개발 제약사항
- **기술 스택**: WebView, React Native, Cocos Creator (게임용)
- **필수**: Granite 기반 스캐폴딩, `@apps-in-toss/framework` 패키지, SDK 2.x (3/23 이후 1.x 불가)
- **번들 크기**: 100MB 제한, `.ait` 패키징
- **디자인**: TDS(토스 디자인 시스템) 필수, 다크모드 미지원, 샌드박스 앱으로만 테스트 가능
- **서버 통신**: mTLS 인증서 필수, 키 발급 7~14 영업일
- **생성형 AI 서비스**: 허용하되 "AI 활용" 고지 및 "AI 생성" 표시 의무 (위반 시 최대 3,000만 원 과태료)
- **파트너 신청**: 사업자 등록 필수 (면세 사업자 불가), 앱 검수 영업일 2~3일

## 2. 수익화 방법

### 실제 파트너사 사례
| 파트너사 | 모델 | 성과 |
|---------|------|------|
| 마나바바 (돌돌디) | IAP + IAA | 월 매출 2.1억원 |
| 디스팟 (할인쿠폰) | CPA 제휴 | 5일 개발, 첫 달 1,000만원 |
| 코심 (eSIM) | 토스페이 직접판매 | 2주 개발, 첫 주 260만원 (광고 없이) |
| 안전집사 (부동산) | 서비스 이용료 | 월 매출 40배 성장 |
| 1인 개발자 | IAA + IAP 혼합 | 미니앱 5개로 월 2,500만원 |

### 토스 자체 광고/수익화 시스템
- IAA: 전면형, 보상형 광고 공식 지원. 보상형이 eCPM 최고
- 마케팅 도구: 앱 푸시, 리워드 프로모션(토스 포인트), 친구 초대, 데이터 대시보드 무상 제공
- 플랫폼 통계: 제휴 미니앱 1,000개 돌파, 75%가 매출 발생, 95%가 현재 운영 중

### 유료 전환 가능 여부
- **가능**. IAP(디지털 콘텐츠)와 토스페이(실물/서비스) 두 가지 결제 수단 지원
- 구독/정기결제: 공식 명시 문서 없으나 토스페이먼츠 빌링키로 기술적 구현 가능 (확인 필요)
- 제한: 금융 상품 중개, 사행성, 디지털 자산 판매 불가

### 계약서 요약기에 적합한 수익화 전략
1. **1순위 — IAA(보상형 광고)**: 무료 분석 3회/일 + 추가 분석 시 광고 시청. 진입 장벽 최저
2. **2순위 — IAP(프리미엄 언락)**: 상세 분석, 비교 분석, 이력 저장 등 프리미엄 기능 유료화 (수수료 20%)
3. **3순위 — IAA + IAP 하이브리드**: 기본 무료(광고) + 프리미엄 유료 조합으로 수익 극대화

## 3. 경쟁사 현황

### 유사 서비스 목록
| 서비스 | 타겟 | 핵심 기능 | 가격 | 한계 |
|--------|------|---------|------|------|
| 앨리비 (BHSN) | B2B 법무팀 | AI 계약서 리뷰, CLM, Legal-OCR | 비공개 | 개인 접근 불가 |
| Law.ai | B2B 기업 | 계약서 검토, 법률자문관리 | 월 2만원/인 (최소 5명) | 월 최소 10만원~ |
| 로폼 | B2B/개인 | 법률문서 자동작성, AI 비교분석 | 연 1.1~50만원 | 문서 작성 중심 |
| 슈퍼로이어 | 변호사 전용 | 법률 리서치, 문서 요약 | 월 9.9~19.8만원 | 일반인 사용 불가 |
| 모두싸인 캐비닛 | B2B | AI 데이터 추출, OCR | 별도 문의 | 전자서명 중심 |

### 차별화 포인트
1. **B2C 공백**: 개인/소규모 사업자용 계약서 분석 앱이 시장에 없음. 유일한 B2C 기회
2. **모바일 퍼스트**: 카메라 촬영 → 즉시 AI 분석. 기존 경쟁사는 PC 웹 기반
3. **무료/저가**: 기존 서비스 월 1~20만원 vs 무료(광고) 또는 건당 소액
4. **앱인토스 선점**: 토스 3,000만 유저 + 법률/계약 카테고리 경쟁 서비스 전무
5. **간편 UX**: CLM/법무관리 없이 "핵심 조항 요약 + 위험 조항 경고" 2가지에 집중
6. **생활밀착형**: 부동산 임대차, 근로계약서, 프리랜서 계약서 등 일반인 빈용 계약서 특화

## 4. 법적 리스크

### 개인정보보호법 관련
- 계약서 이미지를 Claude API(미국)로 전송 = **개인정보 국외이전**에 해당
- Anthropic API는 학습에 데이터를 사용하지 않으므로 **처리위탁**으로 분류 가능성 높음
- 그러나 전송 자체가 국외이전이므로 **동의 의무는 면제되지 않음**
- 필수 고지사항: 이전 항목, 이전 국가(미국), 수령자(Anthropic), 보유기간(7일), 거부 절차
- 민감정보(주민번호, 주소, 계좌번호): 별도 명시적 동의 필요, 주민번호는 원칙적 처리 금지

### Claude Vision API 사용 시 주의사항
- **Anthropic API 정책**: 법률 관련 사용 시 human-in-the-loop 필수, AI 결과를 사람이 작성한 것처럼 표시 금지
- **데이터 보존**: API 로그 7일 보존 후 자동 삭제 (ZDR 계약 시 미저장 가능)
- **변호사법 이슈**:
  - "조항 요약/일반 설명" = 정보 제공 (위반 아닐 가능성 높음)
  - "특정 계약의 유불리 판단 + 수정 권고" = 법률 조언 (위반 가능성 있음)
  - 2025.5 서울행정법원: AI 자동작성은 변호사법 위반 아님으로 판결 (우호적 선례)
  - 무료 서비스일수록 리스크 낮음

### 완화 방법 (1회성 처리, 서버 미저장 등)
1. **개인정보 마스킹 필수**: 전송 전 주민번호, 주소, 계좌번호 자동 마스킹 (OCR + 정규식)
2. **동의 절차 구현**: 업로드 시 국외이전 동의 화면 필수 표시
3. **이용약관/개인정보처리방침**: Anthropic API 사용, 미국 이전, 7일 보존 등 명시
4. **면책 문구**: "법률 조언이 아닌 일반적 정보 제공입니다" + "중요 사항은 변호사 상담 권고"
5. **서비스 범위 제한**: "법적 판단/추론"이 아닌 "조항 요약/일반 설명" 수준으로 한정
6. **ZDR 계약 검토**: Anthropic과 Zero Data Retention 계약 체결 고려
7. **DPA 체결**: Anthropic과 개인정보 처리 위탁 계약서 확보
8. **서버 미저장 설계**: 자체 서버에도 계약서 이미지/분석 결과 저장하지 않는 구조

## 5. 결론 및 PRD 작성을 위한 핵심 결정 사항

### OCR 방식 최종 권장
- **Claude Vision API 직접 사용 권장** (별도 OCR 엔진 불필요)
  - Claude Vision이 이미지에서 직접 텍스트를 인식하고 분석까지 한 번에 처리
  - 별도 OCR(Tesseract 등) + LLM 파이프라인보다 단순하고 정확도 높음
- **마스킹 전처리 필수**: Vision API 전송 전 클라이언트 사이드에서 주민번호 등 민감정보 마스킹
  - 1단계: 클라이언트에서 간단한 OCR(영역 탐지)로 민감 필드 식별
  - 2단계: 해당 영역 블러/마스킹 처리
  - 3단계: 마스킹된 이미지를 Claude Vision API로 전송

### 수익화 전략 권장
- **Phase 1 (출시 초기)**: 완전 무료 + IAA(보상형 광고)
  - 무료 분석 횟수 제한 (예: 일 3회) → 추가 분석 시 보상형 광고 시청
  - 변호사법 리스크 최소화 (무료 = 리스크 낮음)
  - 앱인토스 초기 수수료 면제 혜택 활용
- **Phase 2 (사용자 확보 후)**: IAP 프리미엄 기능 추가
  - 상세 분석 리포트, 분석 이력 저장, 계약서 비교 기능 등
  - IAP 수수료 20% (앱마켓 15% + 토스 5%)
- 구독 모델은 앱인토스 공식 지원 여부 확인 후 검토

### 주요 리스크 및 대응 방안

| 리스크 | 수준 | 대응 방안 |
|--------|------|---------|
| 개인정보 국외이전 | **상** | 동의 절차 + 마스킹 + 이용약관 명시 + DPA/ZDR 계약 |
| 변호사법 위반 | **중** | "정보 제공" 수준으로 한정 + 면책 문구 + 초기 무료 운영 |
| Anthropic 약관 위반 | **중** | human-in-the-loop 안내 + AI 사용 사실 표시 |
| 앱인토스 검수 탈락 | **하** | AI 고지 의무 준수 + TDS 디자인 준수 + 제한 서비스 해당 없음 |
| 수수료 면제 종료 | **하** | 현재 면제 중이나 종료 시점 불명 → 수익 모델에 수수료 반영 설계 |

### 즉시 실행 가능한 사항
- 사업자 등록증 업종이 앱 서비스와 일치하는지 확인 (불일치 시 업종 추가 필요)
- 앱인토스 개발자센터 회원가입 및 워크스페이스 생성 (개발 전에도 사전 등록 가능)
- mTLS 인증서 키 발급 신청 (7~14 영업일 소요 → 조기 착수 권장)
- appName 확정 (등록 후 수정 불가)
