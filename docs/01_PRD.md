## 1. 제품 개요

### 한 줄 요약
계약서 사진 한 장으로 핵심 조항 요약 + 위험 항목 경고를 즉시 확인하는 토스 미니앱.

### 해결하는 문제
- 임대차·근로·프리랜서 계약서를 받았지만 법률 지식이 없어 불리한 조항을 놓치는 개인/소규모 사업자의 문제
- 기존 법률 AI 서비스는 전부 B2B·고가(월 10만원~)이며 모바일에서 즉시 사용 불가
- 토스 3,000만 유저 중 법률/계약 카테고리 미니앱이 **전무** — 선점 기회

### 핵심 가치
| 경쟁사 | 계약서 요약기 |
|--------|------------|
| PC 웹 기반, 파일 업로드 필요 | 스마트폰 카메라로 즉시 촬영 |
| 월 10만원~ (B2B) | 무료 (IAA 광고) |
| 기업 법무팀 전용 | 임차인·알바·프리랜서 누구나 |
| 별도 회원가입 필요 | 토스 계정 자동 연동 |

## 2. 사용자

### 주요 사용자
- **누구**: 20~40대 토스 유저 — 임차인(월세/전세), 취업·아르바이트 지원자, 프리랜서 1인 사업자
- **상황**: 계약서를 받은 자리에서 서명 전 빠르게 내용 확인이 필요할 때
- **목표**: 변호사 상담 없이 계약서의 불리한 조항·누락 항목을 5분 내에 파악

### 사용자 시나리오

**시나리오 1 — 임차인 (핵심)**
1. 집주인에게 임대차 계약서를 받은 세입자가 토스 앱에서 계약서 요약기를 열고
2. 계약서를 카메라로 촬영하면
3. 30초 이내에 핵심 조항 요약·위험 조항·서명 전 체크리스트가 카드 형태로 표시된다

**시나리오 2 — 아르바이트 지원자**
1. 편의점 알바 계약서를 받은 20대가 앱을 열고 계약서를 촬영하면
2. 최저임금 이하 항목, 주휴수당 누락, 불리한 계약 해지 조항이 경고로 표시된다

## 4. 사용자 흐름 (User Flow)

### 핵심 흐름 (Phase 1)
```
앱 오픈 → [최초 1회] 국외이전 동의 → 촬영/갤러리 선택
→ 클라이언트 마스킹 처리 → Flask 서버 → Claude Vision API
→ 분석 완료 → 3카드 결과 화면 → 로컬 저장
```

### 상세 흐름

1. **앱 오픈**: 토스 미니앱 진입 (토스 계정 자동 인증, 별도 로그인 없음)
2. **최초 동의**: 개인정보 국외이전 동의 화면 1회 표시 (Anthropic/미국/7일 보존 명시)
3. **이미지 입력**: 카메라 촬영 버튼 또는 갤러리 선택
4. **마스킹 전처리**: 클라이언트에서 주민번호·계좌번호·상세주소 영역 자동 감지 후 블러 처리
5. **분석 요청**: 마스킹된 이미지를 mTLS 인증 Flask 서버 → Claude Vision API 전달
6. **로딩 화면**: "AI가 계약서를 분석하고 있어요..." (평균 10~30초)
7. **결과 화면**: 3카드 TDS UI 렌더링
   - 카드 ①: 📋 핵심 요약 (임대기간·보증금·월세·특약사항)
   - 카드 ②: ⚠️ 위험 조항 (묵시적 갱신, 수리비 임차인 부담 등)
   - 카드 ③: ✅ 서명 전 체크리스트 (등기부등본 확인, 특약 추가 여부 등)
8. **이력 저장**: AsyncStorage에 세션 ID + 결과 자동 저장
9. **AI 고지**: 결과 하단 "AI가 생성한 정보입니다. 법률 조언이 아닙니다." 고정 표시

## 5. 성공 기준

- [ ] 계약서 촬영 후 결과 표시까지 30초 이내 (Claude API 응답 포함)
- [ ] 임대차 계약서 위험 조항 감지 정확도 80% 이상 (샘플 10건 내부 검증)
- [ ] 국외이전 동의율 85% 이상 (동의 거부 시 서비스 이용 불가 안내)
- [ ] 앱인토스 검수 1회 통과 (재검수 없이)
- [ ] Phase 1 출시 후 30일 내 DAU 500명 달성

## 6. 안 만드는 것 (Out of Scope)

> 이 목록에 있는 건 Phase 1에서 만들지 않습니다.
> AI에게 코드를 시킬 때 이 목록을 함께 공유하세요.

- **법률 조언/법리 해석 기능** — 이유: 변호사법 위반 가능성 (정보 제공 수준으로 한정)
- **계약서 수정 권고 기능** — 이유: 변호사법 위반 직결
- **위험도 점수화 (0~100점)** — 이유: 점수 = AI 판단으로 해석될 소지
- **서버 측 계약서 저장** — 이유: 1회성 처리 원칙, 개인정보 보호
- **근로계약서·프리랜서 계약서 지원** — 이유: Phase 2로 분리
- **IAA 광고 연동** — 이유: Phase 2로 분리 (검수 후 안정화 우선)
- **회원가입/로그인 화면** — 이유: 토스 계정 자동 인증으로 불필요
- **PDF 저장/출력** — 이유: Phase 3 프리미엄 기능
